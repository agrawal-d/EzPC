Instructions to setup automatic judging

  • Run docker run -it agrawald/ezpc-modified
  • Run source /ezpc_dir/EzPC/mpc_venv/activate
  • Download the organizer script using rm -f organizer_script.py && curl -LOJ {{WEBSITE_URL}}/static/contests/scripts/organizer_script.py
  • Learn about how to run the Organizer Script by running python organizer_script.py --help
  • Create a folder for the test dataset. Files must be named 0, 1, 2, ... without extension for each item to be tested.
  • Create a file containing the correct labels ( integer ranging from 0 to number of labels). Each line must contain the correct lable for each testcase. Each label must end be a newline.
  • Run the script! For example, this is one sample command to run it: python organizer_script.py --contest_id 2 --access_key defaultkey --website_url http://127.0.0.1:8000 --num_testcases 3 --labels_file labels.txt --dataset_path dataset --num_classes 5
